## File Type Linux
一切皆文件.
.mp3 .c .h LCD 触摸屏 鼠标 网络文件(socket) 等等 都是文件.



**普通文件**(数据文件)  存数据/程序信息.
- 文本文件
	- 二进制文件

**目录文件**

**设备文件**
操作系统与设备 提供链接的一种文件.
Linux: 把设备当做文件来出来.  操作文件就可以操作设备.
每个设备对应一个设备文件.都在 /dev 目录中


**链接文件** Win 下的快捷方式.
- 软链接 / 符号链接
	- 硬链接


**管道文件**
在进程间传递数据的一种特殊文件.

**套接口文件**
不同计算机之间 网络通信的一种特殊文件.
