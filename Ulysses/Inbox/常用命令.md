## 常用命令
重启机器    shutdown -r now 
切换用户  exit
更换密码 passwd 进账户 直接改密码 

ip 
查询   ip :    addr    查看ip 信息  
加/删 ip :    sudo ip addr add 192.168.0.193/24 dev wlan0
查看   ip :  ip addr  show wlan0  
查看路由器   ip route show
查看 一个ip的路由包 从哪里来的 ip route get 58.217.200.13 (百度的ip)

网络接口类型:
lo  本地回环接口
eth(0-9)  以太网接口
pppx 点对点 接口
