## Linux 入门指南

​ 必须亲自电脑练习.不能只读书.  

​ 尽量用shell,自己写shell script.  

​ 错误解决办法: 先看 shell 的错误提示  → 再看 log 日志文件. → 查询谷歌. 

## 必须习惯:

- 发现问题


- 记录错误信息 及 解决办法


- 定期整理

## 学习:关键是会得『偷』, 偷了会得 改,改了会得发,发则通矣。

## 必会基础技能

- vi 文本编辑器


- shell


- 简单基础命令.
