## brew
> Manual:  wget for example

*查找软件   brew search wget  *            
*已装软件   brew list         *
*安装软件   brew install wget *     
*删除软件   brew remove wget  *
 
 更新brew       brew update    
 更新所有软件   brew upgrade 
 更新指定软件   brew upgrade wget

 软件包信息     brew info wget  
 列出过时软件   brew outdated     
 软件包依赖关系 brew deps wget
