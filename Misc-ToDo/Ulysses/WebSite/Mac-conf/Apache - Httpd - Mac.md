
## Apache / Httpd - Mac 

*Commandes:*
`sudo apachectl -v / start / restart`

**启用Apache :**(不需配置,直接开启就好.)

1. `sudo apachectl start`
	2. 浏览器 → localhost
		3. 出现  It Works!   
			> it works 这个页面在:   
			> `/Library/WebServer/Documents`  
			> 也是 apache的 默认网站路径.

**默认网站目录
`/Library/WebServer/Documents`
