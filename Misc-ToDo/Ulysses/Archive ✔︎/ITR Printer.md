## Printer install

官网下驱动  →  安装驱动  →  接上打印机   → 重启电脑
 
打印机 服务没启动
 
控制面板-安全选项-服务管理-服务-print spool 设置为自动启动.
安装驱动 电脑重启 添加打印机 -选择默认打印机.打印.
