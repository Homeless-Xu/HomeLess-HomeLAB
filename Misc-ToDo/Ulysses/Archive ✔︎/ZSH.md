
## ZSH

** ZSH Function :

- ping -  ✚  ⇥   列出 ping 的所有参数说明.

- ping ✚  ↑      智能匹配有关 ping 的历史命令。

- 双击 Tab        给出所有的补全列表并直接进入到选择列表

- 支持后缀alias,即以什么命令打开特定的后缀名文件。  
	例如alias -s js=mvim,输入hello.js，会以vim打开该文件

**ShortCut**
  
⌘R        → 清屏
^U         → 清行
⌘N/T      → 新 窗口/标签页
⌘;        → 智能匹配历史命令
⌘数字     → 标签页切换
**⌘D**    → 水平分屏 
**⌘⇧D**  → 垂直分屏
⌘⇧h      → 列出剪切板历史
⌘↵       → 全屏切换
