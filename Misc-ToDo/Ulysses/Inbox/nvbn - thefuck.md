
终端下 打错命令
要重新输入么  no!!

输入: fuck  
自动帮你修改成正确命令.

[Github Website][1]


**Requirements**

- python 2.7+
	- pip
	- python-dev


*Mac install*
1. `brew inatall thefuck`

	安装结束会提示你 要把下面一行代码 放到 配置文件中
	> 用 zsh 就放到 .zshrc
	> 用bash 就放到 .bashrc

2. `eval "$(thefuck --alias)"`




*Mac install pip*
`sudo easy_install pip`
